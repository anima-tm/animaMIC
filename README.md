# animaMIC

![animaMIC](doc/img/animaMIC.png)

animaMIC este programul care animă personajul MIC (Muzeul Interactiv al Copiilor) folosind senzori de prezență și servomotoare. Programul rulează pe Arduino Nano R4 și expune o interfață USB pentru comunicație serială.

Pentru detalii, vizitați pagina [Wiki](https://github.com/anima-tm/animaMIC/wiki).
