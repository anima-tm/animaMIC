# animaMIC

![animaMIC](https://github.com/anima-tm/animaMIC/wiki/img/animaMIC.png)

animaMIC este programul care animă personajul MIC al Muzeului Interactiv al Copiilor din Timișoara, folosind senzori de prezență și servomotoare. Programul rulează pe Arduino Nano R4 și expune o interfață USB pentru comunicație serială.

Pentru detalii, vizitați pagina [Wiki](https://github.com/anima-tm/animaMIC/wiki).
